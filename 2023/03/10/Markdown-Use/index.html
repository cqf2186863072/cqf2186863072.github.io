1. markdown基本语法

1.1. 标题

“#”后文本将被设为标题,最多6个”#”,每多一个”#”标题便低一级;

1.2. 换行与分段

空两格一回车换行,空两格两回车另起一段;

1.3. 强调文本

①斜体:”ctrl + i”,在文本两端各加一个星号(shift + 8);
②加粗:”ctrl + b”,在文本两端各加两个星号;

1.4. 列表

①有序列表:”1. “(注意空格);
②无序列表:”*/+/- “(注意空格);

1.5. 插入图片

①复制图片,”ctrl + alt + v”在光标位置插入图片;
②图片文件加入md文件所在文件夹,输入”! + [] + (图片文件名)”;

1.6. 公式

ctrl + m(使用latex公式);

1.7. 表格

默认对齐(左对齐) 居中对齐 右对齐
1 2 3
alt + shift + f可格式化文本编排;

1.8. 链接

①直接黏贴:https://www.bilibili.com/
②选中文本黏贴:bilibili

1.9. 代码块

int a;
a = 100;
